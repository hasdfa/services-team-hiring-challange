## Quick Reference

### Scoring Breakdown

**Total**: 100 Points

- **Code Quality** (30 points) - Clean, maintainable code
- **Architecture** (30 points) - Design decisions, scalability
- **Problem Solving** (20 points) - Edge cases, error handling
- **Documentation** (10 points) - SOLUTION.md quality
- **Engineering Judgment** (10 points) - Trade-offs, tool selection

### Hiring Decision Matrix

| Score Range | Decision        |
| ----------- | --------------- |
| 85-100      | **Strong Hire** |
| 70-84       | **Hire**        |
| 55-69       | **Maybe**       |
| 0-54        | **No Hire**     |
